# 🏥 CareBridge

A responsive healthcare website developed using **HTML, CSS, and JavaScript**. CareBridge provides an intuitive interface for patients to manage appointments, access healthcare resources, interact with an AI-powered assistant, and navigate essential healthcare services.

---

## ✨ Features

- 📱 Responsive design for desktop and mobile devices
- 🤖 AI-powered healthcare assistant interface
- 📅 Appointment booking system
- 👤 User authentication (Login & Profile Setup)
- 🏥 Patient dashboard
- 💳 Payment page
- 👶 Newborn care information
- 🤰 Pregnancy resources
- 🥗 Nutrition guidance
- 📚 Healthcare resources

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
